
# Project Red Pill - Secure AI Chat (Live GPT-4 Connected)

## Contents
- chat.html — Terminal frontend (update fetch URL to your backend)
- api.js — Secure backend API using Node.js and OpenAI
- .env.template — Place your OpenAI API key here

## Setup
1. Deploy api.js on Replit/Vercel
2. Add your OpenAI key to `.env`
3. Point `chat.html` to your live backend URL (replace "your-backend-url")
